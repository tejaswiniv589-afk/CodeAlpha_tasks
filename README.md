# CodeAlpha Python Programming Internship

This repository contains all tasks completed as part of the **Python Programming Internship** at [CodeAlpha](https://www.codealpha.tech). Each folder is a standalone project demonstrating core Python concepts — from control flow and file handling to web scraping and simple rule-based logic.

## Projects

| # | Project | Folder | Key Concepts |
|---|---------|--------|---------------|
| 1 | Hangman Game | `CodeAlpha_HangmanGame` | random, loops, strings, lists |
| 2 | Stock Portfolio Tracker | `CodeAlpha_StockPortfolioTracker` | dictionaries, file I/O, arithmetic |
| 3 | Webpage Title Scraper | `CodeAlpha_WebpageTitleScraper` | requests, BeautifulSoup, file I/O |
| 4 | Basic Chatbot | `CodeAlpha_BasicChatbot` | if-elif logic, functions, loops |

Each project folder contains its own `README.md` with setup and usage instructions.
