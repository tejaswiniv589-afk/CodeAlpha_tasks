# CodeAlpha Basic Chatbot

A simple rule-based chatbot built in Python as part of the **CodeAlpha Python Programming Internship (Task 4)**.

## 📖 About

The chatbot responds to a fixed set of user inputs — greetings, small talk, and farewells — using `if-elif` pattern matching. It's not powered by AI or NLP; every response is a predefined rule.

## 🚀 How to Run

1. Make sure Python 3 is installed.
2. Clone this repository:
   ```bash
   git clone https://github.com/tejaswiniv589-afk/CodeAlpha_BasicChatbot.git
   cd CodeAlpha_BasicChatbot
   ```
3. Run the script:
   ```bash
   python chatbot.py
   ```

## 🎮 How to Use

Just type a message and press Enter. Try things like:

| You say              | Bot responds                                      |
|----------------------|----------------------------------------------------|
| `hello` / `hi`       | A greeting                                         |
| `how are you`        | "I'm fine, thanks! How about you?"                 |
| `what's your name`   | Tells you it's a CodeAlpha chatbot project         |
| `thanks`             | "You're welcome!"                                  |
| `weather`            | Explains it has no live weather data               |
| `bye` / `exit`       | Ends the conversation                              |
| anything else        | Asks you to rephrase                               |

## 🛠 Concepts Used

- `if-elif` conditionals
- Functions
- Loops (`while True` chat loop)
- String handling (`.lower()`, `.strip()`, substring checks)
- `random` (varied greeting responses)
